React Router

The react router enables the creation of multipage apps wothout refreshing the page. It also allows teh use of browser hisory fuctions like using the backbtton, open in new tab, sharing pages, etc that aren't availiable without it

Usage

Set up

Add the roeact router to your project by running npm install react-router-dom@latest in the root directory.

Browser Router

To use the router in an app, import BrowserRouter, Routes, and Route as well as all of the accessable pages to your app jsx file. 

 
            import React from "react";
            import { BrowserRouter, Routes, Route } from "react-router-dom";
            import PageOne from "./PageOne";
            import PageTwo from "./PageTwo";
            import PageThree from "./PageThree";

In the apps return statement, use the <BrowserRouter> to wrap the routes, or pages, in the app. Each route contains properties for the path and element. The path defines the URL of the page while the element propertie defines the componentto be diplayed in the browser. 

        
            function App() {
                return (
                  <BrowserRouter>
                    <Nav />
                    <Routes>
                      <Route path="/" element={<PageOne />} />
                      <Route path="/two" element={<PageTwo />} />
                      <Route path="/three" element={<PageThree />} />
                    </Routes>
                  </BrowserRouter>
                );
              }

Link

For the individual components, import Link from react-router-dom. This is used to create links with in the page that can be accessed by the router. The "to" property in the Link component provides the router with the path it will link to . In the example below, the links direct to "/two" and "/three" which access the matching path from the Route and display the associated element 

        
    import React from "react";
    import { Link } from "react-router-dom";

    function PageOne() {
    return (
        <div>
        <h1>page #1</h1>
        <nav>
            <Link to="/two">go to page 2 </Link>
            <br />
            <Link to="/three">go to page 3</Link>
        </nav>
        </div>
    );
    }
    export default PageOne;
